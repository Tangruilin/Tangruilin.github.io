leveldb 中的迭代器设计

发表于 2025-02-12 更新于 2025-02-16
本文字数： 1k 阅读时长 ≈ 4 分钟

背景

迭代器是访问数据结构的经典设计, leveldb 中为了方便对数据的访问, 设计了诸多迭代器, 针对不同的数据结构进行访问, 本文主要是整理 leveldb 上所有迭代器的设计;

leveldb 迭代器接口

为了方便使用迭代器, leveldb 设计一个迭代器的接口, 接口的 UML 图如下所示.

接口中定义了迭代器常用的, Prev, Next, Seek, Valid 等接口

迭代器的具体实现

leveldb 中的迭代器分为两类:

  1. 底层迭代器: 底层迭代器是一个具备具体功能的迭代器, 通常这类迭代器会用于迭代某个具体的数据结构;
  2. 组合迭代器: 组合迭代器的作用是将两个或者数个底层迭代器组合起来形成一个新的迭代器, 用以完成一个更加复杂的功能;

底层迭代器

MemTableIterator

MemTableIterator 不做过多赘述, 改迭代器主要是遍历一个 skip list.

BlockIter

BlockIter 的核心在于理解 leveldb 中一个 block 的存储方式, 这里会在 leveldb 的文件格式讲解中说明

Version::LevelFileNumIterator

LevelFileNumIterator 主要用于对某一个 level 中的 sst 文件进行访问。该迭代器维护了一个指向文件的下标 index_, 当 index_ 等于文件的总数时表示该迭代器失效, 迭代器指向的 key 和 value 分别表示 key: 该文件中最大的 key 和 value: 该 sst 文件中的 key 数量和文件大小
该迭代器需要关注的核心方法主要是: Next, Prev, Seek

Next

判断当前迭代器有效性。直接增加 index_

Prev

判断当前迭代器有效性。若当前 index 为0则设置为文件个数，表示失效。直接减小 index

Seek(const Slice& target)

通过二分搜索的方式, 拿到该 target 对应的 sst 文件信息

组合迭代器

TwoLevelIterator

TwoLevelIterator 是一个双层组合迭代器, 这个迭代器主要用在两个场景: SST TableIterator 和 ConcatenatingIterator。
SST TableIterator 是单个 SST 文件的迭代器, ConcatenatingIterator leveldb 中一层所有数据的管理模块, 下图简单展示了 ConcatenatingIterator 的分层

TwoLevelIterator 在 TableIterator 中的使用

ldb 文件中的存储格式如下所示:

meta 相关的 Block 属于第三方定制的时候使用的, 在 leveldb 中主要是 布隆过滤器 上会使用, 这里不做赘述;
从存储格式上看, 一个 SST 文件中有 Index Block 和 Data Block, 在 TwoLevelIterator 中的 index_iter 则就是 Index Block 所对应的 iter, Data Block 对应的 iter 是 data_iter。

对应 TwoLevel 迭代器的两层设计, 因此twolevel 迭代器的迭代流程可以如下描述:

  1. 通过 level1 的迭代器, 在大范围中找到对应 key 所在的 block;
  2. 通过 level2 的迭代器, 在更细粒度的范围中找到具体的 key-value;

MergingIterator

MergingIterator 诞生的目的是对一系列的迭代器进行一个有序输出, 这样可以方便 leveldb 对全局的文件进行遍历, 设计这个迭代器的目的则是在 Compaction 以及对 DB 的全局遍历场景中, 可以针对全局中所有的 key 进行遍历。
主要有两个用途：

  1. Compaction——对上下两层文件进行合并，输出一个有序的新文件
  2. DBIter——对全局所有 key/value 存储点进行合并，数据源包括 MemTable/Immutable MemTable/Level 0/Level 1~7 SST，对用户输出一个全局有序的迭代器

这个迭代器中我们需要关注两个函数: FindSmallest 和 FindLargest
基于 FindSmallest 描述函数能力

FindSmallest

这个函数对Iterator 中的所有 children 值进行了比较, 选取 children 中最小的一个, 将该迭代器作为当前迭代器;
